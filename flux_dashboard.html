API Reference

POST endpoints · JSON body · X-API-Key auth

Replace YOUR_MODAL_URL with the endpoint URL from your Modal dashboard.

POST /generate
curl -X POST https://YOUR_MODAL_URL--flux-api-generate.modal.run \ -H "Content-Type: application/json" \ -d '{ "api_key": "sk-lv-YOUR_KEY", "prompt": "kawaii fox girl, pastel sunset", "width": 1024, "height": 1024, "steps": 4, "seed": 42 }'
POST /batch (up to 8 prompts)
curl -X POST https://YOUR_MODAL_URL--flux-api-batch.modal.run \ -H "Content-Type: application/json" \ -d '{ "api_key": "sk-lv-YOUR_KEY", "prompts": [ "neon city at rain", "forest in golden light", "abstract fluid purple" ], "width": 512, "height": 512 }'
Response schema
{ "success": true, "image_b64": "iVBORw0KGgo…", // base64 PNG "width": 1024, "height": 1024, "elapsed_s": 1.34, "model": "FLUX.1-schnell" }
Python client example
import requests, base64 resp = requests.post( "https://YOUR_MODAL_URL--flux-api-generate.modal.run", json={ "api_key": "sk-lv-YOUR_KEY", "prompt": "cinematic product shot, dark background", "width": 1024, "height": 1024, } ) data = resp.json() img_bytes = base64.b64decode(data["image_b64"]) with open("output.png", "wb") as f: f.write(img_bytes)
